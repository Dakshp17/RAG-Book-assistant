# 📚 RAG Book Assistant

A Retrieval-Augmented Generation (RAG) application that allows users to upload PDF documents and ask natural language questions using LangChain, ChromaDB, Hugging Face embeddings, and Mistral AI.

## Features

- Upload PDF documents
- Automatic document chunking
- Semantic search using vector embeddings
- MMR retrieval
- Context-aware AI responses
- Streamlit web interface

## Tech Stack

- Python
- LangChain
- ChromaDB
- Hugging Face Embeddings
- Mistral AI
- Streamlit

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
